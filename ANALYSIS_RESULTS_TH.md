# ผลการอัปเดต v0.11.0

- แก้ Component selector ให้ดึงทุก Component ที่มี Land จาก CAD
- Mapping Table แสดง CAD-only rows และข้อมูล CAD ครบ
- แก้ Excel OOXML โดยย้าย Drawing ไปหลัง Page Setup ตาม schema
- เพิ่ม dimension และแก้ activePane ของ Freeze Pane
- ลดข้อมูลซ้ำใน Zone sheets เมื่อเปิด Compatibility mode
- ทดสอบ Parser กับ CAD 2,591 Components / 24,625 Lands และข้อมูลดิบเดิม 17,662 แถว
